# PWS Config Prototype

People We Support (PWS) configuration prototype for managing care packages, commissioned hours, and occupancy.

## Files

- `PWSConfig.tsx` - Main component
- `pwsConfigData.ts` - Mock data and types

## Features

- Overview tab with metrics and weekly shifts
- Care package management with detailed views
- Occupancy tracking with out-of-service records
- Add new care package wizard
- Out-of-service logging
